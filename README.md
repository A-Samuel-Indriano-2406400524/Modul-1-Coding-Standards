Reflection 1
1. Clean code 
Saya sudah belajar menerapkan dan memahami prinsip clean code dalam pengerjaan kode, 
seperti penamaan variabel yang jelas dan penerapan function yang baik.

2. Security
Saya sudah menerapkan GET hanya untuk mengambil data dan POST untuk mengubah data.

3. Mistakes
Kesalahan yang beberapa kali saya lakukan ketika mengerjakan kode adalah tergesa-gesa untuk
melakukan commit ke Git, padahal masih ada logika yang belum lengkap. Untuk mengatasinya, saya berusaha
untuk lebih hati-hati dengan memeriksa kelengkapan logika dan melakukan bug testing terlebih dahulu sebelum
commit ke Git.
